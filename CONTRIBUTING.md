# Contributing

Chesswright is solo-maintained and still in its early pilot phase (see
the README for what that means in practice).

## Bug reports

These are genuinely the most useful thing you can contribute right now.
Please [open an issue](https://github.com/Hawi254/chesswright/issues/new/choose)
with what you were doing, what you expected, and what happened instead.
No detail is too small, and you don't need a fix in mind.

## Pull requests

Small, focused fixes (a typo, a clear bug with an obvious one-line fix)
are welcome. For anything larger — new features, refactors, UI changes —
please open an issue first to discuss it before writing code. The
project's architecture and priorities are still moving fast enough that
a PR without prior discussion has a real chance of conflicting with
work already in progress, or heading in a direction that doesn't fit
where the project's going.

## What not to expect (for now)

Given this is a single-maintainer pilot project, please don't expect
fast turnaround on reviewing larger PRs, or a roadmap of "good first
issues" to pick from — those will come once the project is further out
of pilot.
